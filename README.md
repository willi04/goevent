# GoEvent Africa

Plateforme de billetterie digitale déployée en production pour la République Centrafricaine.
Accessible sur [goevent.africa](https://goevent.africa)

## Fonctionnalités
- Création et gestion d'événements
- Système de billetterie en ligne
- Authentification utilisateurs
- Interface responsive

## Technologies
- Backend : Python (FastAPI)
- Base de données : PostgreSQL
- Frontend : HTML, CSS (tailwind CSS), JavaScript

## Statut
Application en phase beta – déployée et accessible en ligne pour la République Centrafricaine.
